# 📘 Assignment: Python Data Visualization

## 🎯 Objective

Learn to create data visualizations in Python using libraries such as matplotlib or plotly. Students will build programs that generate various types of charts and graphs to represent data effectively, practicing data manipulation and visualization techniques.

## 📝 Tasks

### 🛠️ Create Data Visualization Program

#### Description

Write a Python program that uses a data visualization library to create and display multiple types of charts from sample data.

#### Requirements

Completed program should:

- Import and use matplotlib or plotly library
- Create a bar chart showing categorical data
- Create a line chart showing trends over time
- Create a pie chart showing proportions
- Save at least one chart as an image file
- Include proper labels, titles, and legends for each chart
